# My Python Projects

This repo contains four tasks inside folders.

## Tasks

Task 1:
 
- Set up with a virtual environment and includes pandas, numpy and matplotlib as dependencies.

Setup

1. Create a virtual environment:  
   `python3 -m venv venv`

2. Activate it:  
   `source venv/bin/activate`

3. Install dependencies:  
   `pip install -r requirements.txt`

4. Run the app:  
   `python main.py`
